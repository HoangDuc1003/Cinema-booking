# 🎬 NitroCine - Movie Ticket Booking App

A complete Full-Stack movie ticket booking application built with the MERN Stack (MongoDB, Express.js, React.js, Node.js). This project integrates a secure authentication system and automated background tasks.

## 🌟 Features
* **User Authentication:** Secure and fast login/registration powered by Clerk.
* **Movie Discovery:** Browse currently showing movies, view details, and watch trailers.
* **Ticket Booking & Seat Selection:** Intuitive seat selection interface with real-time seat status updates.
* **Secure Payment:** Seamless payment gateway integration (Stripe).
* **Automated Background Jobs:** Uses Inngest to handle background events (e.g., sending booking confirmation emails).
* **Admin Dashboard:** Manage movie lists, showtimes, and track user bookings.
